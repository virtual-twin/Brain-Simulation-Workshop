About

In this workshop, participants learn core concepts and methods for personalized whole-brain network modeling using The Virtual Brain (TVB). The workflow combines individualized model definition, simulation, and analysis with ontology-driven model specification (TVB-O) and gradient-based optimization tooling (TVB-Optim) (Martin et al. 2025; Pille et al. 2025).

Getting started

The Python environment is managed with uv:

pip install uv

Tools used:

uv pip install tvb
uv pip install tvbo
uv pip install tvboptim

The full source code lives at https://github.com/virtual-twin/tvb-ontology-optim-workshop.
